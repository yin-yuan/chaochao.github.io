Hexo个人博客搭建

发表于2021-07-15|更新于2021-07-20
|阅读量:

这是一篇记录自己搭建博客的过程 该博客基于Hexo搭建，大家可以学习一下，有写的不好的地方，欢迎在评论区 帮作者指正 谢谢各位了！

搭建博客的前置条件

  1. 注册 GitHub 账号
  2. 安装 Git
  3. 安装 NodeJs

接下来需要将Git和Github进行绑定，后续需要使用Github来托管所搭建的博客，我们需要通过Git将文件上传到Github上，首先得利用SSH登录远程主机 ，登录方式有两种：一种是口令登录；另一种是公钥登录，我们这里使用第二种。

使用git生成公钥

然后复制，当然你也可以通过找到该文件所在的目录下用记事本打开复制，然后登录GitHub，打开Settings,点击 SSH and GPG
